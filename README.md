## Dependencies

- `bash`
- `imagemagick`
- `i3lock-color`

## Installation

1. Install the required dependencies
2. Clone and install

    git clone https://github.com/cherrynoize/n3lock
    cd n3lock
    sudo make install

To uninstall simply run

    sudo make uninstall

## Usage

First cache images with

    n3lock -i PATH/TO/YOUR/IMAGE

From now on you can run `n3lock` like so

    n3lock [-c CONFIG]
